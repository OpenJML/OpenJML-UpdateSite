OpenJML

This page makes available some resources for OpenJML. OpenJML is a set of JML tools built on OpenJDK, and thus updated to Java 1.7.

A summary of the features and the implementation status of OpenJML is here.

Note that OpenJML requires an OpenJDK build of Java 1.7 to run. Such builds can be obtained from this binary snapshot release site. MacOSX releases can be obtained from Dan Zimmerman's site. I (David Cok) am using Dan's builds and this Windows/Cygwin build successfully.

The OpenJML command line tool can be downloaded from here.

The OpenJML user guide is part of the download, but is also available here.

The Update site for the Eclipse plug-in that encapsulates the OpenJML tool is http://jmlspecs.sourceforge.net/openjml-updatesite.

The webpage for the JML project is http://www.jmlspecs.org/index.shtml
