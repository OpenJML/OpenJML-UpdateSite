The OpenJML Eclipse Update Site

This is the Eclipse plug-in update site for the OpenJML Eclipse plug-in.

Within Eclipse (Neon.1 or later) use the Install New Software wizard to add a new site: use the URL of this page, and a title of your choice, such as "OpenJML".

To run this plugin you must be running Eclipse from an OpenJDK build of Java 1.8. See the instructions here.

Problems with or questions about OpenJML can be posted on the GitHub issues site for OpenJML.
